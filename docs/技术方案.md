## 2. API 端点设计

### 认证接口
- `POST /api/v1/auth/register` - 用户注册
- `POST /api/v1/auth/login` - 登录
- `POST /api/v1/auth/refresh` - 刷新Token
- `GET /api/v1/auth/profile` - 获取用户信息

### 数据接口
- `GET /api/v1/trending` - 获取热点列表
- `GET /api/v1/trending/{platform}` - 获取指定平台热点
- `GET /api/v1/history` - 获取历史数据
- `GET /api/v1/keywords` - 管理关键词

### VIP 接口
- `GET /api/v1/vip/status` - VIP状态查询
- `POST /api/v1/vip/pay` - 创建支付订单
- `POST /api/vip/webhook` - 支付回调

### AI 接口
- `POST /api/v1/ai/analyze` - AI分析热点
- `POST /api/v1/ai/translate` - AI翻译

## 3. VIP 权限控制

| 功能 | 免费用户 | VIP用户 |
|------|---------|---------|
| /trending | ✅ | ✅ |
| /trending/{platform} | ❌ | ✅ |
| /history (7天) | ✅ | ✅ |
| /history (30天) | ❌ | ✅ |
| /ai/analyze | ❌ | ✅ |
| 更新频率 | 30分钟 | 5分钟 |

## 4. 技术栈

- **后端**: Python + FastAPI
- **数据库**: SQLite (轻量) / MySQL (生产)
- **缓存**: Redis
- **认证**: JWT
- **部署**: Docker
