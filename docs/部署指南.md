# TrendRadar Mobile 部署配置

## 环境要求

### 服务器
- Ubuntu 20.04+ / CentOS 8+
- 1核 1GB 内存以上
- Python 3.11+
- Node.js 18+ (用于 Flutter Web 构建)

### 安全组端口
- 8000: API 服务
- 80/443: Web 服务

## 快速部署

### 1. API 服务部署

```bash
# SSH 登录服务器
ssh root@your-server-ip

# 安装依赖
apt update && apt install -y python3.11 python3-pip nginx

# 上传项目
cd /opt
git clone your-repo trendradar-mobile
cd trendradar-mobile/api-server

# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
nano .env  # 编辑配置

# 使用 systemd 管理服务
cat > /etc/systemd/system/trendradar-api.service << EOF
[Unit]
Description=TrendRadar API
After=network.target

[Service]
User=www-data
Group=www-data
WorkingDirectory=/opt/trendradar-mobile/api-server
ExecStart=/opt/trendradar-mobile/api-server/venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8000
Restart=always

[Install]
WantedBy=multi-user.target
EOF

# 启动服务
systemctl daemon-reload
systemctl start trendradar-api
systemctl enable trendradar-api
```

### 2. Nginx 配置

```nginx
# /etc/nginx/sites-available/trendradar
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 3. Flutter Web 部署

```bash
# 在本地构建
cd flutter_app
flutter build web --release

# 上传到服务器
scp -r build/web root@your-server-ip:/var/www/trendradar/

# Nginx 添加静态文件服务
```

## Docker 部署（推荐）

```bash
cd api-server
docker-compose up -d
```

## 部署检查

